# 🎭 Digidaw Animasi

**Digidaw Animasi** is a creative graphics and multimedia project that brings the traditional Indonesian folktale **"Bawang Merah Bawang Putih"** to life using the power of code. Developed with **Processing**, this project explores the intersection of storytelling and generative art.

## ✨ Project Highlights

- **Interactive Storytelling**: A digital retelling of a classic tale.
- **Generative Graphics**: Visuals created entirely through code and algorithms.
- **Multi-Scene Experience**: Divided into multiple scenes (Scene 1 to 5) for a structured narrative.

## 🛠️ Built With

- **Language**: Processing (Java-based)
- **Libraries**: Standard Processing graphics libraries

## 📂 Scenes Overview

- `Scene1`: The introduction to the characters.
- `Scene2`: The development of the conflict.
- `scene3`: Key narrative turning points.
- `Scene_5_bawang_merah` & `Scene_5_bawang_putih`: Parallel conclusions for the two main characters.

## 🚀 How to Run

1. Download and install [Processing](https://processing.org/).
2. Open each scene folder in the Processing IDE.
3. Press the **Run** button to view the animation.
